Spam Email Classifier using NLP

 Project Overview
This project classifies email/SMS messages as Spam or Not Spam using Natural Language Processing (NLP) and Machine Learning techniques.

 Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorization
- Naive Bayes Classifier

 Dataset
SMS Spam Collection dataset containing labeled spam and ham messages.

 Approach
- Text preprocessing
- TF-IDF feature extraction
- Naive Bayes classification
- Model evaluation using accuracy and confusion matrix

 Results
- Accuracy achieved: ~98%

 How to Run
```bash
pip install -r requirements.txt
python spam_classifier.py
